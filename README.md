# plagiarism_checker
plagiarism checker using python nltk

# Installation
- git clone https://github.com/sundar2001/plagiarism_checker.git
- cd plagiarism_checker
- pip3 install -r requirements.txt

# Usage
- python3 plagiarism_checker.py
